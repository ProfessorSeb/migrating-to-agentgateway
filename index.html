Why Migrate?

AgentGateway OSS is a Linux Foundation project that unifies LLM routing, MCP tool aggregation, and agent-to-agent connectivity in a single binary.

⚡

Rust Performance

Written in Rust. Lower latency, lower memory footprint. Runs on 128MB RAM vs LiteLLM's 8GB recommendation.

🔗

Unified Gateway

LLM proxy + MCP gateway + A2A in one binary. No more stitching together separate tools for each protocol.

🔒

Enterprise Security

JWT auth, CEL-based authorization, prompt guards, PII detection, and per-tool access control out of the box.

⚙

Kubernetes Native

Built on Gateway API with CRDs, xDS dynamic config, and Helm charts. No Python runtime in your cluster.

📈

OpenTelemetry Built-in

Native OTLP tracing, metrics, and structured access logs. No callback plugins needed.

🔄

Drop-in Compatible

Same OpenAI-compatible API on the same port. Your client code doesn't change - just swap the proxy.
